AI Setup · One Time
Enable AI Features
The Anthropic API blocks direct browser calls for security (CORS policy). You need to add one small proxy file to your Vercel project. It takes about 2 minutes.
Step 1 — Create the proxy file
In your Vercel project, create a file at: /api/ai.js
// /api/ai.js — paste this into your Vercel project export default async function handler(req, res) { res.setHeader('Access-Control-Allow-Origin', '*'); res.setHeader('Access-Control-Allow-Methods', 'POST,OPTIONS'); res.setHeader('Access-Control-Allow-Headers', 'Content-Type'); if (req.method === 'OPTIONS') return res.status(200).end(); const response = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': process.env.ANTHROPIC_API_KEY, 'anthropic-version': '2023-06-01' }, body: JSON.stringify(req.body) }); const data = await response.json(); res.status(response.status).json(data); }
Step 2 — Add your API key to Vercel
Go to your Vercel project → Settings → Environment Variables → Add:
ANTHROPIC_API_KEY = sk-ant-...

Get your key at: console.anthropic.com
Step 3 — Redeploy & activate
Push to Vercel (redeploy). Then click the button below to switch 3401 to use your proxy.
